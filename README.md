# React Upvoty integration

> Simple wrapper for integrating Upvoty widget into react applications

## Example

```javascript
import UpvotyWidget from 'react-upvoty';

const Upvoty = () => (
  <UpvotyWidget
    ssoToken={'123'}
    baseUrl={'123'}
    boardHash={'123'}
  />
);
```

---

## Installation

```shell
$ yarn add react-upvoty
$ npm install react-upvoty --save
```
